üìñ Introduction

Welcome to the Windsurf Privacy Toolkit!

This comprehensive toolkit gives you complete control over Windsurf's access to your system. Whether you're concerned about privacy, want to understand what your AI coding assistant can access, or need to manage permissions for security testing, this toolkit has you covered.

üéØ What This Toolkit Does

‚úÖ Monitor & Control

  • View all macOS system permissions (Camera, Microphone, Full Disk Access)
  • See which folders Windsurf has accessed
  • Track permission changes with timestamps
  • Analyze security risks

üîí Lock Down Access

  • Lock Windsurf to Documents folder only
  • Remove access to specific locations
  • Clean up stale folder tracking
  • Export chat history before removing access

üß™ Test & Reset

  • Reset all TCC permissions
  • Clear folder access tracking
  • Launch Windsurf in sandbox mode
  • Force permission re-requests

üöÄ Getting Started

  1. Launch the Toolkit: Double-click the Desktop app or run python3 windsurf_privacy_gui.py
  2. Start with Dashboard: Get an overview of your current privacy status
  3. Explore Each Tab: Each tab focuses on a specific aspect of privacy control
  4. Take Action: Use the buttons to scan, remove, lock, or reset as needed
üí° Pro Tip: Start by running the Audit tab to get a complete picture of what Windsurf has access to, then use the other tabs to make changes.

‚ö†Ô∏è Important Notes

Before You Begin:
  • This toolkit modifies Windsurf's databases and system permissions
  • Always backup important data before making changes
  • Some operations require sudo/admin access
  • Windsurf may need to be restarted after changes
  • Chat exports are saved to _Chatlogs folders (never pushed to git)

üìã System Requirements

  • OS: macOS 10.13 or later
  • Python: Python 3.7 or later
  • Dependencies: PyQt6 (installed automatically)
  • Permissions: Admin access for some features

üîí Permissions

Overview

The Permissions tab shows all macOS system permissions that Windsurf has requested or been granted. This helps you understand what system resources Windsurf can access.

Permission Types

üì∑ Camera & Microphone

What it allows: Access to camera and microphone hardware

Why Windsurf needs it: For video calls, screen sharing, or audio input features

Privacy impact: High - can record audio/video

Recommendation: Only grant if you use these features

üíæ Full Disk Access

What it allows: Read/write access to all files on your system

Why Windsurf needs it: To access files in protected locations

Privacy impact: Critical - can access everything

‚ö†Ô∏è High Risk: Full Disk Access gives complete file system access. Only grant if absolutely necessary.

ü§ñ Automation & AppleScript

What it allows: Control other applications via AppleScript

Why Windsurf needs it: To integrate with other development tools

Privacy impact: Medium - can control other apps

üìÅ Folder Access

What it allows: Access to specific folders (Desktop, Documents, Downloads)

Why Windsurf needs it: To open and edit files in these locations

Privacy impact: Medium - limited to specific folders

Checking Permissions

  1. Click üîç Scan Permissions
  2. Review the permissions table
  3. Check status: ‚úÖ Granted, ‚ùå Denied, ‚ö†Ô∏è Not Requested

Permission Status Meanings

  • ‚úÖ Granted: Windsurf has this permission
  • ‚ùå Denied: You've explicitly denied this permission
  • ‚ö†Ô∏è Not Requested: Windsurf hasn't asked for this yet
  • üîÑ Pending: Permission request is waiting for your response

Managing Permissions

To change permissions:

  1. Open System Settings ‚Üí Privacy & Security
  2. Select the permission type (e.g., "Full Disk Access")
  3. Find "Windsurf" in the list
  4. Toggle on/off as needed
  5. Restart Windsurf for changes to take effect
üí° Minimal Permissions: For best privacy, only grant permissions you actually need. Most users don't need Camera, Microphone, or Full Disk Access.
